# TaskFlow Backend

Express + MongoDB backend for TaskFlow.

## Features

- Register API
- Login API
- JWT Authentication
- Add Todo
- Get Todos
- Delete Todo

## Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Setup Instructions

1. Clone repository
2. Run npm install
3. Create .env file
4. Run npm run dev
